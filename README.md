# VisionScript
A scripting language that uses pure math to create a powerful rendering engine.

## Examples

- Detailed parametric equation:
    ```
    n = [1...1000]^2
    P(t) = sum((cos(n*t), sin(n*t))/n)
    parametric P(2*pi*t)

    position = (-pi^2/12, 0.05)
    scale = (20.0, 20.0)
    ```
